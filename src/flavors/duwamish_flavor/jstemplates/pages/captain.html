First make sure you're signed in and have requested access to be an administrator.

The sign in button is on the top right

If you're having trouble signing in send a message to feedback@heyduwamish.org and we'll fix it right away!

Then request access

Let us know you want to be an admin for Hey Duwamish so you can start editing the map.

Once you've been approved as an admin

You can now edit a site on the map

Open a site that you have permission to edit, and click "Edit this post". If you don't see an "Edit this post" button, you are either not signed in or you don't have permission to edit that particular site.

Hit the Save button to save your changes. Hit the Edit this post button to cancel your changes and go back. Hit the Remove button to remove the site from the map.

You can also add new featured sites

  1. Make sure you're signed in.
  2. Hit "Add a report" as if you were adding a community report.
  3. Select the option to "Add a featured site"

Draw your site on the map

You can add a marker and select a marker symbol, draw a line, or draw a polygon.

Compose your content

The content editor offers you a variety of formatting options such as:

  • Header selection for large or mid-sized page headers
  • Images and video embeds (Youtube or Vimeo)
  • Rich text bold, italics, underline, and color selection
  • Hyperlinks to other web pages

Hit the Save button if you're ready to publish. Otherwise, you can store it as a draft by setting the state to unpublished at the top of the editor.

Have fun creating your community map!
